# Contexto del producto

## 1) Descripción de las 3 páginas a construir

### Página 1: Inicio / Exploración
Muestra una propuesta clara de valor de la plataforma, un buscador principal (destino, fechas y huéspedes), categorías destacadas de alojamientos y recomendaciones iniciales para inspirar al usuario.

### Página 2: Resultados de búsqueda
Muestra el listado de alojamientos disponibles según filtros y criterios de búsqueda. Incluye tarjetas con imagen, precio, ubicación, valoración y servicios clave, además de controles para ordenar y filtrar resultados.

### Página 3: Detalle del alojamiento
Muestra la información completa de una propiedad: galería de fotos, descripción, amenities, reglas, reseñas, ubicación y desglose de precio. Incluye el módulo para seleccionar fechas y confirmar una reserva.

## 2) Listado de componentes

- Header / Navbar
- Hero de búsqueda
- Formulario de búsqueda (destino, fechas, huéspedes)
- Chips de categorías
- Grid de tarjetas de alojamientos
- Tarjeta de alojamiento
- Panel de filtros
- Selector de orden
- Paginación o carga incremental
- Galería de imágenes
- Bloque de descripción del alojamiento
- Lista de amenities
- Módulo de reseñas
- Mapa / bloque de ubicación
- Card de reserva (fechas, precio, total)
- Footer

## 3) Usuario objetivo y su objetivo en la plataforma

La plataforma está pensada para una persona que quiere planificar un viaje de forma rápida y confiable, comparar opciones sin fricción y reservar un alojamiento que se ajuste a su presupuesto, fechas y preferencias de experiencia.
